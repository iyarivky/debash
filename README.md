<h1 align="center">DeBash</h1>
<h3 align="center">online tool that aims to deobfuscate bash scripts using JavaScript</h3>

## Usage
- Go to https://debash.deno.dev
- Input your obfuscated bash string
- Press "Deobfuscate" button
- Done
